## Projeto desenvolvido com Java e Spring Boot

Habilidades desenvolvidas no projeto
- JAVA
- POO
- VSCode
- GIT e GITHUB
- PADRÃO REST
- API's
- JSON
- MONGO DB ATLAS
- DEPLOY HEROKU
